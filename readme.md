##C++大作业之学生选课系统

* 配置教程：[传送门](http://www.cnblogs.com/joeblackzqq/p/4332945.html)
* 数据库建表：运行仓库中的course_choice.sql，将自行建表。

###2、已完成部分：
* 学生、教师、管理员的登录。
* 查看课程安排。
* 学生选课、查看已选课程（含成绩）。

###3、待完成部分：
* 用户界面编写（使用QT）。
* 各模块联调。
* 测试与优化

###4、注意事项：
* 各变量名的命名规范：
	* 类名一律大写。如:class MYSQL{}
	* 单单词变量名小写。如：index
	* 多单词变量名小写，各单词间以下划线连接。如：is_login
* 开发中对已有的表务必保持各表明属性名的一致，以便最后的联调。
